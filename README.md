> basic ruby tdd setup using rspec

## Installation
```
Pull down the repository 
```
Install [Yeoman](http://yeoman.io)

```bash
npm install -g yo 
```

Within the repository, run:
```
npm link
```
navigate to empty folder and run:

```bash
yo rubytdd
```
